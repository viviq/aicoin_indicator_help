1. 快速入门

简介

通过 AICoin Script 你能操作AICoin 内部丰富的数据，计算你想要的指标，并将其绘制到图表分析。

快速上手

我们通过一个MACD的指标进行讲解：

// @version=2

// 定义指标相关参数
N1 = 12
N2 = 26
M = 9

// 计算指标的值
dif = ema(close, N1) - ema(close, N2)
dea = ema(dif, M)
macd = (dif - dea) * 2

// 计算预警条件的值
golden_cross = crossup(dif, dea)
dead_cross = crossdown(dif, dea)
cross_up_zero = crossup(dea, 0)
cross_down_zero = crossdown(dea, 0)

// 定义预警预警条件，以便在添加预警的窗口中使用
alertcondition(golden_cross, title='MACD(12,26,9)金叉')
alertcondition(dead_cross, title='MACD(12,26,9)死叉')
alertcondition(cross_up_zero, title='MACD(12,26,9)上穿零轴')
alertcondition(cross_down_zero, title='MACD(12,26,9)下穿零轴')

// 绘制数据到图表
plot(dif, title='DIF', color='rgba(47,210,178,1)', lineWidth=1, lineDash=[0])
plot(dea, title='DEA', color='rgba(255,180,0,1)', lineWidth=1, lineDash=[0])
plotColumn(macd, title='MACD')

赋值语句

定义一个数值为12的变量：

N1 = 12

使用nan赋值一个空值：

N1 = nan

同样，我们可以将函数的结果赋值给变量：

// 单个返回值
ema12 = ema(close, 12)

// 多个返回值
[dif, dea, macd] = macd(close, 12, 26, 9, 'EMA', 'EMA')

进阶用法

变量引用

// @version=2
a = close[1]

使用条件语句

a = nan
b = nan
if (high > close && high > open){
    a := close
    b := open
}
plot(a)
plot(b)

使用循环语句

// @version=2
a = nan
for(i=0;i<10;i++){
    a := i;
}

使用函数定义

// @version=2
func_ma(x,y) {
    sum = 0.0 ;
    for (i = 0; i < y; i++){
        sum := sum + x[i]/y ;
    }
    return sum ;
}
func_ma10 = func_ma(close, 10)

使用多数据源函数

// @version=2
// 获取对应交易对的close数据
security('ethswapusdt:okcoinfutures', '5m', close)

2. 行情数据函数

基础行情数据

名称 说明 类型
open 当前K线的开盘价 数值序列数据
high 当前K线的最高价 数值序列数据
low 当前K线的最低价 数值序列数据
close 当前K线的收盘价。在实时监控中，收盘价即为最新价。 数值序列数据
hl2 (最高价 + 最低价) / 2的快捷运算 数值序列数据
hlc3 (最高价 + 最低价 + 收盘价) / 3的快捷运算 数值序列数据
ohlc4 (开盘价 + 最高价 + 最低价 + 收盘价) / 4的快捷运算 数值序列数据
volume 当前K线的成交量 数值序列数据
time 当前K线的时间 数值序列数据

高级行情数据（需要@version=2）

名称 说明 限制条件
timeclose 当前K线的收盘时间 语法版本需要 @version=2
timestart 实时运行开始的K线事件 语法版本需要 @version=2
vbuy 资金流向买量 限PRO、胜率、信号会员可用
vsell 资金流向卖量 限PRO、胜率、信号会员可用
oi_close 持仓量 语法版本需要 @version=2
liq_short_size 买入（平空）张数 限PRO、胜率、信号会员可用
liq_long_size 卖出（平多）张数 限PRO、胜率、信号会员可用
注意：
  • 数值序列数据可以进行算术运算，运算的结果仍然是数值序列数据
  • 数值序列数据可以进行关系运算，运算的结果是布尔序列数据，可以作为预警条件的数据源
  • 更多使用方法可以参考内置的脚本样例

3. 指标函数

MACD指标

macd(source, N1, N2, M, maType1, maType2)

参数说明：

名称 类型 说明
source 序列数据 如close，更多取值见行情数据
N1 数值 短周期
N2 数值 长周期
M 数值 DEA周期
maType1 字符串 均值的方式，支持 MA/EMA/SMMA
maType2 字符串 均值的方式，支持 MA/EMA/SMMA

返回值：dif、dea、macd

例子：

[dif, dea, macd] = macd(close, 12, 26, 9, 'EMA', 'EMA');

移动平均线

MA - 简单移动平均线

ma(source, length)

返回简单移动平均值（即算术平均值）

EMA - 指数平滑移动平均线

ema(source, length)

返回系数alpha为2/N+1的指数移动平均值

布林带

boll(source, N, M)

返回布林带指标，包括中轨、上轨和下轨

返回值：boll、ub、lb

KDJ指标

kdj(source, length, mac1, mac2, maType)

返回KDJ随机指标的K、D、J值

RSI指标

rsi(source, length)

返回相对强度指数

其他指标

  • td(source) - 狄马克序列
  • obv(length, maType) - 能量潮指标
  • stochrsi(source, rsilength, stochlength, fast, slow, maType) - 随机相对强弱指数
  • cci(source, length, maType) - 商品路径指标
  • skdj(source, length, mac1, mac2, maType) - 慢速随机指标
  • bbi(source, N1, N2, N3, N4, maType) - 多空指数
  • supertrend(factor, atrPeriod) - 超级趋势指标（需要@version=2）
  • alligator(source1, source2, N1, N2, N3, N4, N5, N6, maType) - 鳄鱼指标（需要@version=2）

4. 计算函数

统计函数

  • highest(source, length) - 计算周期内最高值
  • lowest(source, length) - 计算周期内最低值
  • max(source1, source2, ...) - 获取多个值中的最大值
  • min(source1, source2, ...) - 获取多个值中的最小值
  • cum(source) - 计算累积和
  • sum(source, length) - 计算周期和
  • abs(source) - 计算绝对值
  • sd(source, length) - 计算标准差
  • dev(source, length) - 计算平均偏差

逻辑函数

  • na(source) - 判断是否为非有效数字
  • nz(source, source1) - 用指定数替换NaN值
  • cross(source1, source2) - 判断是否交叉
  • crossup(source1, source2) - 判断是否上穿
  • crossdown(source1, source2) - 判断是否下穿
  • ref(source, length) - 引用周期前的数据

数学函数（需要@version=2）

  • log(number) - 自然对数
  • log10(number) - 常用对数
  • sqrt(number) - 平方根
  • sin(angle) - 正弦函数
  • cos(angle) - 余弦函数
  • tan(angle) - 正切函数
  • round(number) - 四舍五入
  • pow(base, exponent) - 幂运算

时间函数（需要@version=2）

  • second(time, timezone) - 获取秒
  • minute(time, timezone) - 获取分
  • hour(time, timezone) - 获取小时
  • dayofmonth(time, timezone) - 获取每月日期
  • dayofweek(time, timezone) - 获取每周日期
  • month(time, timezone) - 获取月份
  • year(time, timezone) - 获取年份
  • weekofyear(time, timezone) - 获取周数

5. 画图、预警与条件表达式

画图函数

plot - 绘制线图

plot(series, title, color, linewidth, lineDash, display, offset, showLast, trackPrice, excludeRange)

plotColumn - 绘制柱状图

plotColumn(series, title, color, fill, display, offset, showLast, excludeRange)

plotText - 绘制文字（需要@version=2）plotText函数参数不要跨行跨行。

plotText(cond, title, text, refSeries, bgColor, color, fontSize, placement, display, offset, showLast, excludeRange)

plotShape - 绘制图形（需要@version=2）

plotShape(cond, title, refSeries, shape, color, fill, placement, display, offset, showLast, excludeRange)

fill - 填充背景

fill(series1, series2, color, display)

预警函数

alertcondition - 创建预警条件

alertcondition(source, title, direction)

参数说明：

  • source - 布尔类型的序列数据
  • title - 警报条件的标题
  • direction - 警报条件的方向，取值为buy或sell

条件表达式

三元运算符

condition ? expr1 : expr2

例子：

// 如果是阳柱，赋值 1 否则赋值 -1
a = close > open ? 1 : -1;

if-else语句

if(condition){
    expr1;
} else {
    expr2;
}

for循环（需要@version=2）

for (init; condition; post) {
    statement;
}
注意：循环数量限制为500次，若单次循环超过500则报错

8. 多数据源函数

security函数

从另外一个数据源中获取数据：

security(symbol, period, expr)

参数说明：

  • symbol - 请求的数据标识
  • period - 请求的数据周期
  • expr - 需要计算和返回的表达式

获取其他数据的symbol值

方法1：直接复制标识码

在PC端中，找到交易对/指数后方的复制标识码按钮，点击后即可复制当前交易对的symbol值。

方法2：按格式使用指数指标

交易对和指数
// 获取对应交易对的close数据
security(syminfo.tickerid, '1m', close)

// 获取OKX-BTC多空持仓人数比指数的close数据
security('i:lsprbtc:okex', '5m', close)
持仓数据
// 获取ethusdt永续的持仓量
security("interest:ethswapusdt:okcoinfutures", '5m', close)
资金费率
// 获取ethusdt永续的资金费率
security('fundrate:ethswapusdt:okcoinfutures', '5m', fundingRate)
多空持仓人数
// 获取eth的多空持仓数据
security('lspr:ethswapusdt:okcoinfutures', '5m', ratios)
精英持仓
// 获取合约精英趋向指标的值
security('elite:mkrswapusdt:binance', '', buyAccount)

// 获取合约精英持仓比例指标的值
security('elite:mkrswapusdt:binance', '5m', buyInterest)

9. Version2版本新特性

主要改进

  • 赋值语法：初次赋值使用 =，再次赋值使用 :=
  • 定义域处理：使用if、for、func等定义的内部变量不能在结构外使用
  • 数组处理：支持数组初始化，使用下标{}访问
  • 函数处理：支持递归函数和自定义函数
  • 语法简化：可以省略语句结尾的分号
  • 条件绘图：支持使用三元运算符对color进行赋值
  • 数据类型扩展：支持int、float、string、bool、byte、any类型

使用限制

  • plot和alert中仅支持普通表达式的计算，不支持函数调用或if等语句
  • 三元运算符的表达式不支持变量的定义，可以使用变量赋值
  • indicator函数仅可使用一次
  • plot和alert不能定义在局部定义域内，如if、for、func等
  • 序列变量的储存值只有500个周期，所有的历史数据引用最大值为499
  • 数组的初始化最大值为1000
  • 除法运算中，若除数为0，则返回空值或nan

循环控制

支持break和continue语句：

for i=0;i<10;i++{
    if i == 5 {
        break
    }
    // 只输出0,1,2,3,4
}

for循环增强

支持range函数遍历数组：

a = [1,2,3,4,5]
for i in range(a){
    // 输出数组索引: 0,1,2,3,4
}
